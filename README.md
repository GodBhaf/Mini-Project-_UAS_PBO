# Mini-Project-_UAS_PBO
# Sistem Pemesanan Makanan

## Deskripsi
Sistem Pemesanan Makanan adalah aplikasi berbasis Java yang digunakan untuk mengelola proses pemesanan di restoran. Program ini memungkinkan customer memilih menu, memesan makanan, menghitung total pembayaran, dan memproses pembayaran tunai.

## Fitur
- Menampilkan daftar menu makanan.
- Menyimpan data customer dan nomor meja.
- Memproses pesanan makanan.
- Menghitung total pembayaran.
- Menghitung kembalian pembayaran.
- Menangani error input dengan exception handling.

## Struktur Program
UAS_PBO/
│
├── Restoran.java       (Class utama program)
├── Menu.java          (Data menu makanan)
├── Order.java         (Data pesanan customer)
├── Person.java        (Abstract class induk)
├── Customer.java      (Data customer)
├── Payment.java       (Abstract class pembayaran)
└── CashPayment.java   (Pembayaran tunai)

## Konsep OOP
Class dan Object
Encapsulation
Inheritance
Polymorphism
Abstraction
Abstract Class
Exception Handling

## Cara Menjalankan Program
1. Pastikan JDK telah terinstal.
2. Compile seluruh file javac *.java
3. Jalankan program java Restoran
